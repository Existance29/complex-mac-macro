Keyboard
=====

Functions
----------------

keyDown(key)
"""""""""

Description:  

Presses the key down

Parameters:  

* key: key to press (see keyboard keys)

Example:

.. code:: python

  from pymacauto import *

  keyDown("a") #press down the a key

keyUp(key)
"""""""""

Description:  

Release the key

Parameters:  

* key: key to press (see keyboard keys)

Example:

.. code:: python

  from pymacauto import *

  keyUp("a") #releases the a key
